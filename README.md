# Assignment 1:- Corrected Python Code
Description: A collection of Python code snippets that have been corrected from errors.

Example 1: Incorrect code
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

Corrected code
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)
